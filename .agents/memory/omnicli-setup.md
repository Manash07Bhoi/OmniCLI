# OmniCLI setup lessons

## Database must be migrated on first boot
Run `pnpm --filter @workspace/db run push` once after install or the API returns 500 ("no such table: notes"). This is not automatic — task #4 proposes fixing it.

**Why:** The drizzle.config.ts uses `~/.local/share/omni/omni.db` (XDG_DATA_HOME). Fresh envs have no DB, Drizzle creates the file but not the tables until `push` is called.

**How to apply:** Any fresh Replit clone or fork needs `pnpm --filter @workspace/db run push` before starting the API server.

## @workspace/api-zod must be a dashboard devDependency
Without it in `artifacts/omni-dashboard/package.json`, pnpm doesn't symlink it and Vite can't resolve the import at dev time.

**Why:** pnpm workspace symlinks are only created for declared dependencies. The dashboard imports `DevHashInputAlgo`, `DevJsonInputAction` from `@workspace/api-zod` directly.

## Generated enum types already exist — don't duplicate
`lib/api-zod/src/generated/types/` (generated by Orval) already exports `DevHashInputAlgo`, `DevJsonInputAction`, `BackupSnapshotStatus`, `ModuleStatus`. Adding them again in a hand-written file causes TS2308 ambiguity errors.

## archive.tsx / search.tsx: `enabled` query option needs `as never` cast
The Orval-generated `UseQueryOptions` type requires `queryKey`, which we don't have client-side. Cast the options object `as never` (pattern already used in search.tsx line 21).

## omni-config Clippy: rename from_str → from_ext
`ConfigFormat::from_str` triggers `clippy::should_implement_trait` because it matches the `std::str::FromStr` trait name. Renamed to `from_ext` in `omnicli/crates/omni-config/src/formats.rs`.

## BLAKE3 in API server is a SHA-256 stub
`artifacts/api-server/src/routes/dev.ts` lines 28-35: when algo === "blake3", it falls back to SHA-256 silently. The CLI binary computes real BLAKE3. Task #3 tracks fixing this.

## Three workflows are the services
- `artifacts/api-server: API Server` — Express, port 8080, serves /api
- `artifacts/omni-dashboard: web` — Vite React, serves /
- `artifacts/mockup-sandbox: Component Preview Server` — serves /__mockup

## Rust CLI build
`cd omnicli && cargo build --release` — takes ~9min on first run (Replit cold), ~9s incremental. Binary at `omnicli/target/release/omni`.
